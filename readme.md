# Add To Home Screen

Add to home screen (A2HS): Easy way to add 'install app' for your websites.

```
Laravel 6+    
```

## Commands
```bash
php artisan a2h:install
```

### Option for all the commands
`--force` This will override the existing file, if it exists.
